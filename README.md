# Blast
Building a multiplayer game

How to run:

    LINUX
        
        if you use CLion, copy these directories to cmake-build-debug or cmake-build-release
            - textures
            - sounds
            - shaders
            - server_config.txt
            
        
        
        if you dont, just run these commands:
        
        git checkout main
        cmake .
        make
        ./BLAST

    
    Windows
        Just use CLion, and do the same as in linux
